## Larizo Perfume Website

A modern, responsive website concept for **Larizo Perfume**, built as a practice project with real-world application in mind. This project focuses on creating a clean, brand-focused UI and a smooth user experience.

### Features
- Responsive design (mobile, tablet, desktop)
- Product showcase pages
- Clean and elegant UI
- Optimized performance

### Tech Stack
- Next.js, TailwindCSS
