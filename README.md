# Larizo — Perfume Brand Website

A modern, responsive website built for **Larizo**, a real perfume brand based in Pakistan. Developed as a practice project with a live pitch opportunity in mind — if accepted, this becomes a production deployment.

---

## Live Demo
🔗 [larizo.vercel.app](https://larizo.vercel.app)

---

## Features

- Full-screen hero section with cinematic layout
- Scent Families section — educates visitors on fragrance notes
- Product collections grid with gender filter badges
- Slide-out cart drawer with quantity management
- Shared layout underline animation on navbar (Framer Motion)
- Navbar synced with active section on scroll
- Smooth scroll powered by Lenis
- Mobile hamburger menu with fullscreen dropdown
- Fully responsive — mobile, tablet, desktop

---

## Tech Stack

| Technology | Purpose |
|---|---|
| Next.js 14 (App Router) | Framework |
| Tailwind CSS v4 | Styling |
| Framer Motion | Animations |
| Lenis | Smooth scrolling |
| Lucide React | Icons |
| Context API | Cart state management |
